# 📊 Social Media Insights Dashboard

A web-based analytics dashboard for aggregating and visualizing cross-platform social media performance data across Facebook, Instagram, and TikTok.

Built as a practical tool for digital marketing teams to consolidate reporting from multiple platforms into a single, interactive interface — eliminating manual spreadsheet work.

---

## Features

- **Multi-platform support** — Import and analyze data from Facebook, Instagram, and TikTok in one place
- **Interactive charts** — Dynamic visualizations powered by Chart.js
- **File uploads** — Drag-and-drop CSV/Excel data ingestion (PapaParse + SheetJS)
- **User authentication** — Secure login system via Supabase
- **Clean reporting UI** — Designed for non-technical marketing users

---

## Built With

- HTML, CSS, JavaScript
- [Chart.js](https://www.chartjs.org/) — data visualization
- [PapaParse](https://www.papaparse.com/) — CSV parsing
- [SheetJS](https://sheetjs.com/) — Excel file handling
- [Supabase](https://supabase.com/) — authentication and backend

## Background

This dashboard was developed to support digital operations work managing 40+ social media accounts across multiple clients, where manual cross-platform reporting was a recurring bottleneck.
